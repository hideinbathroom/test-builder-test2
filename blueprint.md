
# Lotto Number Generator

## Overview

This project is a simple web application that generates random lotto numbers for the user.

## Plan

1.  **HTML (`index.html`):** Create the basic structure of the web page, including a title, a button to trigger the number generation, and a container to display the generated numbers.
2.  **CSS (`style.css`):** Style the elements to make the interface clean and user-friendly.
3.  **JavaScript (`main.js`):** Implement the logic for generating 6 unique random numbers between 1 and 45. The generated numbers will be displayed on the page when the user clicks the button.
